# Welcome to the Veridian Wallet User Guide

This guide provides a comprehensive overview of the Veridian Wallet and its core functionalities.
Here, you’ll find clear explanations of the app’s features, along with step-by-step tutorials and animations to help you navigate, set up, and use the wallet with confidence.

The Veridian Wallet is designed as the world’s first KERI-based mobile wallet, offering a secure, decentralized way to manage your digital identity.
Whether you’re setting up your wallet for the first time, managing identifiers and credentials, or exploring advanced features like group multi-signature flows, this guide will walk you through the process with accessible, easy-to-follow instructions.

Let’s get started!
